The Website
The website contains information about the programming languages C# and Python, and about the special features these languages have.
People who want to know a bit more about these languages can visit the site.
At the end, they can also vote and see which language is more popular.

Technologies used on the site and their roles
- html - the content and structure of the site
- css - the design and look of the site
- javascript - user interactions and data
- c# - the server

Pages -

Home page (home.html) - a page that explains a bit about the site
- Description of the technologies and their uses
- The site's purpose and a summary about it

Python page (python.html) - a page that explains Python
- The page will have some general information and explanation about Python
- Lists of special/unique things about Python
- An example of Python code (as an image)
- Link to the official Python website

C# page (c#.html) - a page that explains C#
- Information about C#
- An example of C# code (as an image)
- A table of various operations and how to do them in C#

Vote page (vote.html) - on this page you can vote for which language is better and see other votes
- There will be a short explanation about the vote
- The page has a C# button that, when clicked, increases a server-side variable counting votes for C#
- The page also has a Python button that increases a server-side variable counting votes for Python
- there is also a delete button if someone wants to remove their vote and then they can vote again
- The page also shows the number of people who voted for each one

On every page there will be an arrow to go to the next or previous page (or both).
All pages will also share a common nav bar for those who don't want to use the arrows.
